Basic structure of HTML

HTML is structured in two parts which are the Head and the Body. The contents the title of the website. Contents in the Head structure does not appear on the website page but rather on the the browser tab, because of this feature the head section is used to title the website. The body section contains all the information that will appear on the website page itself.

Steps in creating a table in HTML

Just like in normal life where we create tables to help organize our information, we could also create tables in HTMl to help in organising data/information on the website. The only difference between creating a table on paper and on a website is that on paper the table has lines/borders but on a websites they don't. Follow the steps below to create a table in HTML

  1. Create a Table using <table></table> tag
  2. Within the table tag create a table row using <tr></tr> tag
  3. within the table row tag add the table headings using <th> tag
  4. Now you can add data to the table using the <td> tag

If you follow the steps above correctly you will get something like this

Cars Model Colour
Benz C 300 White
BMW X6 Blue
Kia Picanto Red
Tesla Model 3 Black
